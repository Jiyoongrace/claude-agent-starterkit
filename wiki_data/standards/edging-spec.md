# Edging 기준 명세

## 개요

**Edging**은 열연 공정에서 슬라브의 폭 방향 끝단을 압연하여 치수를 정확하게 맞추는 공정입니다.
Edging 기준값 변경은 후공정 전체에 파급 효과를 미칩니다.

## 현행 기준값

| 항목 | 현행값 | 허용 범위 |
|---|---|---|
| Edging 압하량 | 25mm | 15 ~ 35mm |
| 두께 허용 공차 | ±0.3mm | ±0.5mm 이내 |
| 폭 허용 공차 | ±2mm | ±5mm 이내 |

## 파급 효과 분석

Edging 기준 변경 시 아래 후공정에 연쇄 영향이 발생합니다:

```
Edging 기준 변경
  ├── 압연 하중 증가 (Rolling Load ↑)
  │     ├── 냉각속도 재계산 필요
  │     └── 롤 교체 주기 단축
  └── 표면 품질 등급 변동 가능성
        └── 설비 유지보수 일정 재조정 필요
```

## 변경 절차

1. 시뮬레이터 에이전트로 파급 효과 사전 분석
2. 품질관리팀 검토 (최소 2영업일)
3. `TB_EDGE_SPEC` 테이블 업데이트
4. 연관 서비스 재배포 확인
