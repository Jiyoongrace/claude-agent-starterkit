# DG320 에러 가이드

## 에러 개요

**DG320**은 열연 공정에서 슬라브(Slab)의 **단중(Unit Weight)**이 허용 범위를 초과할 때 발생하는 에러입니다.

> **단중**: 제품 단위 길이당 무게(kg/m). 압연 후 최종 제품의 품질 기준이 됩니다.

## 발생 조건

| 파라미터 | 정상 범위 | DG320 발생 임계값 |
|---|---|---|
| 폭 (Width) | 900 ~ 1200mm | > 1180mm (두께 8.5mm 기준) |
| 두께 (Thickness) | 6.0 ~ 12.0mm | > 9.5mm (폭 1200mm 기준) |
| 타겟 단중 | 200 ~ 295 kg/m | > 295 kg/m |

## 원인 분석

1. **슬라브 설계 파라미터 초과** — 폭 또는 두께가 허용 범위를 벗어남
2. **단중 허용 범위 상수 변경** — `TB_SLAB_TARGET` 테이블의 상수값 오류
3. **압연 하중 계산 오류** — Edging 기준 변경 후 재계산 미반영

## 조치 방법

```
1. TB_SLAB_TARGET 테이블에서 타겟 단중 허용 범위 확인
2. 폭/두께 파라미터를 안전 범위로 조정 (시뮬레이터 에이전트 활용)
3. Edging 기준값 최신화 여부 확인
4. 조치 후 Job 재실행
```
