# 제조 현장 인텔리전스 플랫폼 — 위키

이 위키는 철강/열연 공장 운영에 필요한 표준, 서비스 명세, 운영 절차서를 포함합니다.

## 디렉토리 구조

| 폴더 | 내용 |
|---|---|
| `standards/` | Edging 기준, 에러 코드 가이드 등 기술 표준 문서 |
| `services/` | MSA 각 서비스 API 명세 |
| `runbooks/` | IRMS 등록, 권한 신청 등 운영 절차서 |
